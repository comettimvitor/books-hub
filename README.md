# Books‑Hub - README

Books‑Hub é um projeto desenvolvido como atividade substitutiva da
Pós‑Tech. Trata-se de uma aplicação escrita em Java, com suporte a
Docker, criada para gerenciar livros e servir como base de estudos em
backend.

Como rodar localmente: 1. Clone o repositório: 

    git clone https://github.com/comettimvitor/books-hub.git cd books-hub

Rodar com Docker: 

    docker-compose up –build

A Collection para teste se encontra dentro do projeto na pasta *Postman Collections*
